Platform is dead

Due to almost zero interest, the common project hub/platform idea was discontinued, and existing services decomissioned. Here's some tools we collected that might help you run your OSS project (in no particular order):

Continuous Integration

  • Travis CI
  • Semaphore
  • CircleCI
  • drone.io
  • CodeBetter
  • AppVeyor (Windows)

Code Reviews

  • Reviewable
  • GerritHub

Communication

  • Gitter
  • Slack
  • HipChat
